

# KingBurguer

É um aplicativo de comida onde o usuário pode se cadastrar e fazer login.

O usuário pode selecionar uma comida, ver detalhes, gerar cupons e acompanhar cupons ativos e expirados.

---

## 🛠 Tecnologias Utilizadas


- Kotlin
- Jetpack Compose
- MVVM
- Navigation Compose
- Material 3
- Retrofit
- Hilt
- DataStore
- Splash Screen
- Coil
- Material
- Dager Hilt
